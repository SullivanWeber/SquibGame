<h1>SquibGame</h1>
  <p>Authors: Sullivan Weber & Zach Jameson</p>
  
  This was a project created for Iowa State's Problem Solving in Software Engineering final project. 

  SquibGame is a game coded in C that consist of multiple minigames in the format of Squid Game on Netflix. The game was made to learn about C syntax and basic logic in programming. It was compiled using Cygwin.
